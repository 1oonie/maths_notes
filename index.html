What is a matrix?

A matrix is a rectangular array of elements. The dimensions of a matrix with m rows and n columns is written as m × n .

There are two special matrices. The null matrix is an m × n matrix where all the items are 0. It is the additive identity for the set of all m × n matrices. The identity matrix m × m matrix and is the multiplicative identity for m × m matrices.

The transpose of matrix A is denoted by A T . To tranpose a matrix you must swap the rows and the columns. This means that the dimensions of the transposed matrix will be n × m if the original matrix had a dimension of m × n .

Matrix multiplication

Matrices can be multiplied by a scalar value. This is trivial as all you have to do is multiply each item of the matrix by a scalar.

Unfortunately, multiplying a matrix by another matrix is not as easy. For two matrices to be conformable for multiplication the number of columns of the left matrix must equal to the number of rows of the right matrix. That is to say that if the left matrix has a dimension of m × n and the right matrix has a dimension of s × p , s must equal n for the matrices to be conformable for multiplication.
