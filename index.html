Chapter 1: Matricies

Last updated: 13 Oct 2024

What is a matrix?

A matrix is a rectangular array of elements. The dimensions of a matrix with m rows and n columns is written as m×n.

There are two special matricies. The null matrix is an m×n matrix where all the items are 0. It is the additive identity for the set of all m×n matricies. The identity matrix m×m matrix and is the multiplicative identity for m×m matricies.

The transpose of matrix A is denoted by AT. To tranpose a matrix you must swap the rows and the columns. This means that the dimensions of the transposed matrix will be n×m if the original matrix had a dimension of m×n.

Addition and subtraction of matricies

To add two matricies together, they must be the same dimension.

Matrices that have appropriate dimensions for an operation are said to be conformable.

Matrix multiplication

Matricies can be multiplied by a scalar value. This is trivial as all you have to do is multiply each item of the matrix by a scalar.

Unfortunately, multiplying a matrix by another matrix is not as easy. For two matricies to be conformable for multiplication the number of columns of the left matrix must equal to the number of rows of the right matrix. That is to say that if the left matrix has a dimension of m×n and the right matrix has a dimension of s×p, s must equal n for the matricies to be conformable for multiplication.

Unlike the multiplication of real numbers, matrix multiplication is not commutative: AB does not necessarily equal to BA. When AB does equal to BA, the two matricies are said to commute.
